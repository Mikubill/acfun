# acfun-uploader

Simple AcFun Video Uploader

上传视频到AcFun的小工具

## usage

```shell
./acfun-uploader [options] file(s)

  -token string
    	Your User Token (a.k.a acPasstoken)
  -uid string
    	Your User ID (a.k.a auth_key)
  -verbose
    	Verbose Mode
```
